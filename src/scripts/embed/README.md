# paratii-player2
Second iteration of Paratii Player

## Prerequisites

* node >= 8.9.0


## Installing

    $ yarn install


## Testing

None yet!


## Development

Build the client

    $ yarn run build-client

Or, build the client and rebuild on file changes:

    $ yarn run build-client:dev


Boot up the server

    $ yarn run start

Then visit `localhost:3000/video/:video_id`.
